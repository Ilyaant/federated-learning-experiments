# Эксперименты на датасете Brodatz
В этой ветке находится код для обучения различных архитектур нейронных сетей на текстурном датасете Brodatz.
## Используемые модели
На данный момент сравниваются следующие архитектуры:
- EfficientNet
- ConvNext
## Запуск эксперимента
Для запуска эксперимена с выбранной моделью из списка выше выполнить:
```bash
python run_experiment.py --model=<model_name>
```
